		\textbf{Question}:

		\noindent Find the values of $x,y,z$ so that the vectors $\myvec{x\\2\\z}$
		%$x\hat i + 2\hat j + z\hat k$
		and $\myvec{2\\y\\1}$ 
		%$2\hat i + y\hat j + \hat k$
		are equal.

		\textbf{Solution:} From the given information,


		\begin{align}
			\myvec{x\\2\\z} = \myvec{2\\y\\1}
		\end{align}
		
		\begin{align}
			\Rightarrow x = 2, y = 2, z = 1
		\end{align}
